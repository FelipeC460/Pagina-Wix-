Librerias
En C++, la biblioteca estándar es una colección de clases y funciones, escritas en el núcleo del lenguaje. Algúnas importantes como < iostream > es la que se encarga de la salida y entrada de información; < time.h > nos permite tener acceso al reloj interno del computador y utilizarlos; librería < string >, nos permite utilizar cadena de caracteres; < conio > & < windows > nos permite manipular los colores y las ventanas de mi computador(básicamente el tema estético).

Funciones
Una función es un bloque de código que realiza alguna operación. Una función puede definir opcionalmente parámetros de entrada que permiten a los llamadores pasar argumentos a la función. ... Una función también puede devolver un valor como salida.
(Anclaje-Volver)
Variables
Es un "cajón" en el cual podemos meter y almacenar datos, existen 5 tipos de variables, la variable (int) que es una variable tipo entero, la variable (char) la cuál nos sirve para almacenar una letra o símbolo, la variable (string) nos sirve para guardar palabras, la variable (float) esta nos sirve para guardar números decimales y por último, la variable (bool) que nos sirve para almacenar números binarios o booleanos.
(Anclaje-Volver)
Condicionales
Los condicionales en C++, son una estructura de control esencial al momento de programar y aprender a programar. Un condicional, permite establecer una serie de condiciones al interior de nuestro programa, que nos ayudan a determinar que acciones llevará cabo dadas ciertas circunstancias
(Anclaje-Volver)
Tipos de Condicionales
Los condicionales if, son una estructura de control condicional, también llamadas estructuras selectivas de casos simples (porque solo definen un posible flujo), lad cuales nos permiten tomar cierta decisión al interior de nuestro algoritmo
(Anclaje-Volver)
Tipos De Condicionales 2
Los condicionales Switch, son una estructura de control condicional, que permite definir múltiples casos que puede llegar a cumplir una variable cualquiera, y qué acción tomar en cualquiera de estas situaciones,
(Anclaje-Volver)
Vectores
El tipo de variable vector permite almacenar varios valores bajo una sola variable y realizar operaciones complejas con esta. Todos los valores dentro del vector han de ser del mismo tipo (todas int o todas string o todas char o todas bool).
(Anclaje-Volver)
Ciclos
Un ciclo o bucle permite repetir una o varias instrucciones cuantas veces lo necesitemos, por ejemplo, si quisiéramos escribir los números del uno al cien no tendría sentido escribir cien líneas mostrando un numero en cada una, para esto y para muchísimas cosas más, es útil un ciclo, permitiéndonos hacer una misma tarea en una cantidad de líneas muy pequeña y de forma prácticamente automática.
